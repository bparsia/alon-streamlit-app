# ALOn Model Explorer

Interactive web application for editing, visualizing, and checking formulae against ALOn (Action Logic with Opposing (for n agents)) models.

ALOn is discussed in:

* Baltag, Alexandru, Ilaria Canavotto, and Sonja Smets. ["Causal agency and responsibility: a refinement of STIT logic."](https://link.springer.com/chapter/10.1007/978-3-030-53487-5_8) In Logic in high definition: Trends in logical semantics, pp. 149-176. Cham: Springer International Publishing, 2020.
* Canavotto, Ilaria. [Where Responsibility Takes You: Logics of Agency, Counterfactuals, and Norms.](https://link.springer.com/book/10.1007/978-3-031-17111-6) Springer Nature, 2022. 

We draw exclusively on chapter three of Canavotto's book though there are but minor notational and expositional differences between the two sources.



## Features

We allow specification of a certain, very restricted class of ALOn models by press ganging Mermaid class diagrams plus YAML front matter. We can then do a full" responsibility analysis on that model at the distinguished index `m/h1`. That is, for each agent and group (`A`), an action performed by A at `m/h1` (`act/A`), and for a given outcome (`p`) which is true at the successor of `m/h1` we determine whether:

1. Actual causation: `but(act/A, q)`, `ness(act/a, q)`
2. Deliberative xstit: `[A dxstit]q`
3. Causal responsibility: `[A pres]q`, `[A sres]q`, `[A res]q`

We determine these via a translation to another logic (primarily OWL 2 DL) and running a reasoner on the result.

## Quick Start

You can see this app at []()

Or run it locally.


```bash
# Clone the repository
git clone https://github.com/your-username/alon-streamlit-app.git
cd alon-streamlit-app

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run streamlit_app/app.py
```

The app will open at http://localhost:8501

You need to install the Konclude reasoner in an appropriate location. 


## Technical Details

- **Language**: Python 3.9+
- **Framework**: Streamlit plus the [streamlit-mermaid](https://github.com/neka-nat/streamlit-mermaid) plugin
- **Parser**: [Lark](https://github.com/lark-parser/lark) (for ALOn and Mermaid grammars), [strict-yaml](https://github.com/crdoconnor/strictyaml/tree/master) (for frontmatter parsing)
- **Reasoner**: [Konclude](http://konclude.com) (OWL 2 DL reasoner)
